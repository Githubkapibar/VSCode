gulp 4

  • Скачиваем и устанавливаем Node
  • Скачиваем и устанавливаем Git
  • Проверка версии node --version, npm --version, npx --version
  • Устанавливаем gulp глобально: npm install --global gulp-cli
  • Проверяем версию gulp --version
  • Создаём package.json (описывает установленые пакеты): npm init
  • Проверяем версию gulp --version

Tasks gulp

  • Создаём файл gulpfile.js
  • Создаём структуру сайта(папки)
  • нельзя называть функции и таски одинаково

    • Структура gulp task: function nameTask(done) { done(); } exports.default = nameTask; - запустит таск по умолчанию при команде gulp exports.nameTask = nameTask; - запустит таск по названию: gulp nametask

При работе с разными технологиями не забываем добовлять синтаксис: html, css, sass, gulp ...

Плагины Gulp

  • Rename: npm i gulp-rename --save-dev(rename = require('gulp-rename'))
  • Sass: npm i gulp-sass --save-dev(sass = require('gulp-sass')
  • Autoprefixer: npm i gulp-autoprefixer --save-dev(autoprefixer = require('gulp-autoprefixer')
  • Concat: npm i gulp-concat --save-dev
